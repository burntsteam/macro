# ContentSearch

Search for items by their content. For documents, this searches the document body text. For emails, this searches the email message body. For chats, this searches the message content. This tool finds items based on what's inside them, not their titles or names.

## Parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `entityTypes` | array | Yes | Which types of items to search. Leave empty to search all types. Examples: ['documents'], ['emails', 'documents'], ['channels'] |
| `query` | string | Yes | The text content to search for. This searches within the body of documents, emails, and messages. |
