Search is the easiest way to find specific files in Macro. When searching for multiple terms, use quotation marks for an exact match. Otherwise, the default is to match each term independently.

> Use the <kbd>/</kbd> shortcut to open up search or the <kbd>cmd</kbd>\+<kbd>k</kbd> menu to search by name.

Use the filter button in the upper right corner to narrow results by type (agent results, messages, etc.). To see a result preview in search, use <kbd>l</kbd> to expand and <kbd>h</kbd> to close.
