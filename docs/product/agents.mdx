> To create a new chat use shortcut <kbd>c</kbd> \+ <kbd>a</kbd>

Agents have access to everything in your workspace giving them, and you, full context into your workspace. In the agents tab, you can see every chat that you have had with your agent and can continue where you left off.

> Use <kbd>l</kbd> and <kbd>h</kbd> to preview different chat results and <kbd>j</kbd> and <kbd>k</kbd> to toggle up and down through them.

Ask for summaries, clarifications, etc and the agent will produce answers using all the information it has in the workspace.

### Automations

Agents are also capable of automations/cron jobs. For instance, daily reminders or weekly summaries can be added inside the agents module. The responses from automations will populate here, in the agents tab and in your inbox.

> Use shortcut <kbd>g</kbd> \+ <kbd>a</kbd> to go to agents

To create an automation, go to the agents module and toggle to the automations tab. Select create an automation and fill out the requirements. You can @mention channels, docs, people, etc inside of the automations form.

Each time the automation is run, the result will appear in your inbox. For example, a daily task todo cron job will appear in your inbox at the time specified for it to run each day.
