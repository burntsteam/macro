### **Global**

<kbd>c</kbd>  Open create launcher

<kbd>cmd</kbd>\+<kbd>k</kbd>  Open command menu   

<kbd>cmd</kbd>\+<kbd>\\ </kbd>  Create new split   

<kbd>cmd</kbd>\+<kbd>;</kbd>  Toggle settings

<kbd>cmd</kbd>\+<kbd>z</kbd>  Undo

<kbd>shift</kbd>\+<kbd>cmd</kbd>\+ <kbd>z</kbd> Redo

 <kbd>j</kbd> / <kbd>k</kbd>  Navigate (vim-style)

 <kbd>enter</kbd> Open in current split

 <kbd>shift</kbd>\+<kbd>enter</kbd>  Open in new split

 <kbd>escape</kbd>  Close launcher

### **Create Launcher**

<kbd>c</kbd> \+ <kbd>d</kbd>   Create doc

<kbd>c</kbd> \+ <kbd>t</kbd>   Create task

<kbd>c</kbd> \+ <kbd>e</kbd>  Create email

<kbd>c</kbd> \+ <kbd>m</kbd>  Create message 

<kbd>c</kbd> \+ <kbd>a</kbd>  Create AI chat 

<kbd>c</kbd> \+ <kbd>n</kbd>  Create canvas

<kbd>c</kbd> \+ <kbd>f</kbd>  Create folder

<kbd>c</kbd> \+ <kbd>o</kbd>  Create code file

### **List Navigation (Tasks, Inbox, etc.)**

<kbd>j</kbd>  / <kbd>↓</kbd>  Move down

<kbd>k</kbd> / <kbd>↑</kbd>  Move up                 

<kbd>h</kbd> / <kbd>←</kbd> Collapse item

<kbd>l</kbd> / <kbd>→</kbd>  Expand item 

<kbd>shift</kbd>\+<kbd>j</kbd> / <kbd>shift</kbd>\+<kbd>↓</kbd>  Select down 

<kbd>shift</kbd>\+<kbd>k</kbd> / <kbd>shift</kbd>\+<kbd>↑</kbd>  Select up               

<kbd>gg</kbd>  Jump to top  

<kbd>shift</kbd>\+<kbd>g</kbd> Jump to bottom       

<kbd>x</kbd>  Toggle select item      

<kbd>cmd</kbd>\+<kbd>a</kbd>  Select all             

 <kbd>tab</kbd> Next tab

<kbd>shift</kbd>\+<kbd>tab</kbd>  Previous tab     

<kbd>1</kbd> –<kbd>9</kbd> Switch to tab by number

<kbd>cmd</kbd>\+<kbd>f</kbd>  Search in list

<kbd>escape</kbd>  Clear selection

### **Entity** **Actions**

<kbd>e</kbd>  Mark done          

<kbd>delete</kbd> / <kbd>backspace</kbd>  Delete item(s) 

<kbd>r</kbd>  Rename  

<kbd>cmd</kbd>\+<kbd>d</kbd>  Duplicate 

<kbd>m</kbd>  Move to folder

<kbd>shift</kbd>\+<kbd>cmd</kbd>\+ <kbd>c</kbd>  Copy link          

### Tasks

<kbd>shift</kbd>\+<kbd>cmd</kbd>\+ <kbd>b</kbd>  Copy branch name   

<kbd>shift</kbd>\+<kbd>cmd</kbd>\+ <kbd>o</kbd>  Open property editor

<kbd>shift</kbd>\+<kbd>cmd</kbd>\+ <kbd>p</kbd>  Set priority 

<kbd>shift</kbd>\+<kbd>cmd</kbd>\+ <kbd>a</kbd>  Set assignee       

<kbd>shift</kbd>\+<kbd>cmd</kbd>\+ <kbd>s</kbd>  Set status

### **Channels & Messaging**

<kbd>↑</kbd> / <kbd>↓</kbd> Previous / next message 

<kbd>shift</kbd>\+<kbd>g</kbd> Jump to latest message  

<kbd>enter</kbd>  Reply to message        

<kbd>e</kbd>  Edit message          

<kbd>backspace</kbd> Delete message 

<kbd>cmd</kbd>\+<kbd>f</kbd>  Find in channel   

<kbd>escape</kbd>  Clear selection 

### **Markdown Editor**

<kbd>cmd</kbd> \+<kbd>b</kbd>  Bold

<kbd>cmd</kbd>\+ <kbd>i</kbd>  Italic 

<kbd>cmd</kbd>\+<kbd>u</kbd>  Underline

<kbd>shift</kbd>\+<kbd>cmd</kbd>\+<kbd>x</kbd>  Strikethrough

<kbd>shift</kbd>\+<kbd>cmd</kbd>\+<kbd>h</kbd>  Highlight 

<kbd>cmd</kbd>\+<kbd>e</kbd> Inline code

### **Canvas**

<kbd>v</kbd>  Select tool

<kbd>h</kbd>  Hand (pan) tool

<kbd>r</kbd>  Shape tool

<kbd>x</kbd>  Connector tool

<kbd>p</kbd>  Pencil tool

<kbd>t</kbd>  Text tool

<kbd>cmd</kbd>\+<kbd>=</kbd> / <kbd>cmd</kbd>\+<kbd>-</kbd>  Zoom in / out

<kbd>cmd</kbd>\+<kbd>g</kbd>  Group selection

<kbd>shift</kbd>\+<kbd>cmd</kbd>\+<kbd>g</kbd>  Ungroup

<kbd>]</kbd> / <kbd>\[</kbd>  Bring to front / send to back

<kbd>↑</kbd> <kbd>↓</kbd> <kbd>←</kbd> <kbd>→</kbd>  Nudge selection (<kbd>shift</kbd> for larger steps)

### **Split Layout / Window**

<kbd>cmd</kbd>\+<kbd>escape</kbd> Close split / go home    

<kbd>shift</kbd>\+<kbd>escape</kbd>  Maximize split (spotlight)

<kbd>shift</kbd>\+<kbd>h</kbd>  Focus split to the left

<kbd>shift</kbd>\+<kbd>l</kbd>  Focus split to the right
